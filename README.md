# cokicat's dotfiles

Here are my dotfiles for a simple and lightweight desktop.

- Window Managers: JWM, Fluxbox
- Icon Theme: [Papirus](https://www.pling.com/p/1166289)
- Mouse Theme: [Bibata Modern Ice](https://www.pling.com/p/1197198)
- Shell: bash
- Terminal: kitty
- Editors: nano, micro
- Task Manager: htop
- Compositor: picom
